# Global E-Commerce Performance & Revenue Analysis

An end-to-end data analytics solution designed to transform raw e-commerce data into dynamic, executive-level insights. This project showcases a full data pipeline workflow—combining database manipulation via SQL with visual storytelling and data modeling in Power BI.

---

## 🛠️ Tech Stack & Skills Demonstrated
* **Database Querying:** SQL (Data extraction, filtering, and aggregation)
* **BI Tool:** Power BI Desktop
* **Data Modeling:** Star Schema / Fact & Dimension relationships
* **Calculations:** Advanced DAX (Custom business measures and metrics)
* **UI/UX Design:** High-contrast dark theme optimized for stakeholder readability

---

## 🔍 Core Insights & Visual Features

### 1. Temporal Trend Analysis
* Built a continuous, chronological timeline chart tracking monthly revenue fluctuations to isolate peak sales seasons.
* Organized month-of-year sorting logic to ensure logical, sequential reporting.

### 2. Geographic & Operational Distribution
* Leveraged dynamic map boundaries to display revenue concentrations across global regions.
* Tracked payment method preferences (Card, COD, Wallet) alongside product category performance using custom-colored bar and donut charts.

### 3. Key Performance Indicators (KPIs)
* Modeled high-level summary metrics focusing on Total Revenue, Average Customer Value, Total Orders, and supply chain fulfillment metrics (Average Delivery Days).

### 4. User Interactivity
* Implemented custom button-tile slicers across geographic regions, allowing stakeholders to seamlessly filter the entire reporting interface with a single click.
